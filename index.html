Description of the project

We are a two members team, in the second year of a Degree in Video Game Design and Development. As a project for the Game development subject we had to create a 2d platformer. The objective of the game is to finish both maps and collect all the coins you can! Be carefull, you have 3 lives, if you lose those you die.

The Legend of The Boy

Pol Galan Morales

Pau Raurell Gomis

Game Controls


A: Move Left
D: Move Right
Space: Jump
N: Attack
M: Dash

Debug Funcionalities

F1: Start the first level
F2: Start the second level
F3: Restart the current level
F5: Save
F6: Load.
F9: Show the colliders and logic
F10: Activate/Deactivate the god mode
F11: Toggle 30fps/60fps
